# Sunset RoadTrip
Ride accross the desert with a 4x4.

### Setup
```bash
git clone git@github.com:SabrinaNedjah/Sunset-Roadtrip.git
```

Then open the `index.html` file in the `/src` folder.

### Features
- Day / Night mode (touch the sky to enable/disabled)

### Technologies
- HTML
- CSS
- JAVASCRIPT
